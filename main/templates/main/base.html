О нас

Изготавливаем домики для кошек и котов на заказ по индивидуальному чертежу или эскизу, а также по стандартному проекту с учетом пожеланий клиента. Дом для питомца будет служит игровой площадкой и удобным местом для отдыха. Мы изготовим такой комплекс, который не будет занимать много места и не будет выделятся из интерьера квартиры.
